# RIF Unlocker (Flask)

موقع ويب بسيط شبيه بـ https://s-unlocker.com باستخدام Flask.

## المتطلبات
- Python 3.8 أو أحدث
- Flask

## التثبيت والتشغيل

1. تثبيت الحزم:
   ```bash
   pip install -r requirements.txt
   ```
2. تشغيل السيرفر:
   ```bash
   python app.py
   ```
3. افتح المتصفح على:
   [http://127.0.0.1:5000](http://127.0.0.1:5000)

## التخصيص
- عدل منطق المعالجة في `app.py` حسب الخدمة المطلوبة.
- عدل التصميم في `static/style.css` وواجهة المستخدم في `templates/index.html`.
